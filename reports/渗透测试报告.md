# 渗透测试报告 - 网络安全实验

**作者**：赵诚信  
**日期**：2025

## 1. 测试目标
校园模拟网络环境（路由器、Web 服务、数据库）

## 2. 测试工具
- Kali Linux
- Wireshark
- sqlmap
- Nmap

## 3. 发现的漏洞
- SQL 注入漏洞
- XSS 跨站脚本攻击
- DDoS 潜在风险

## 4. 攻击演示与防御
- **SQL注入**：使用 sqlmap 探测 → 防御：使用参数化查询
- **XSS**：注入恶意脚本 → 防御：输入过滤 + CSP
- **DDoS**：流量洪泛模拟 → 防御：限流 + CDN

## 5. 总结与收获
通过本次实验，深入理解了 OWASP Top 10 漏洞，掌握了基本渗透测试流程和安全防御意识。
